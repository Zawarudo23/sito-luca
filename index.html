Educazione Civica

Con l’avvento di Internet e delle tecnologie, la navigazione nel mondo digitale è ormai diventata parte integrante delle nostre vite, e, come tale, è importantissimo che vi siano dei diritti e anche dei doveri, al fine di garantire una pacifica convivenza (anche se online) tra gli utenti. I diritti sono fondamentali perché garantiscono la libertà in tutte le sue forme. Ad esempio, su Internet ognuno può esprimere la propria opinione, purché non vada ad intaccare la libertà e i diritti altrui. Grazie all'educazione civica possiamo conoscere i nostri diritti e doveri e ci permette di rispettarli in maniera adeguata e, contemporaneamente, di rispettare gli altri.

Come ho realizzato questo Sito

Questo sito è stato realizzato attraverso tre linguaggi di programmazione: HTML, CSS, JavaScript.

HTML, o HyperText Markup Language, consente agli utenti Web di creare e strutturare pagine web attraverso sezioni, paragrafi e collegamenti utilizzando elementi, tag e attributi. Tuttavia, vale la pena notare che l'HTML non è propriamente un linguaggio di programmazione in quanto non può creare funzionalità dinamiche, Viene, infatti, denominato linguaggio di markup.

CSS è un linguaggio per specificare come i documenti (solitamente strutturato utilizzando un linguaggio di markup, come HTML, XML o SVG) vengono presentati agli utenti: come sono stilizzati, strutturati, ecc. CSS è, quindi, utilizzato per modificare e migliorare l'aspetto grafico di un sito web. Le classi CSS sono usate per specificare attributi grafici come font, dimensione, colore, spaziatura, bordo e posizione degli elementi all'interno di una rappresentazione web.

JavaScript è un linguaggio di scripting o di programmazione che consente di implementare funzionalità complesse sulle pagine Web, visualizzando aggiornamenti tempestivi dei contenuti, mappe interattive, animazioni 2D/ Grafica 3D, jukebox video a scorrimento, ecc.
